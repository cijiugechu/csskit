# Coverage

Comprehensive test coverage datasets; predominantly CSS sources from the internet.

## Purpose

This directory is used as a repository of CSS files for various libraries to validate code against.

## Contents

- [`css-tokenizer-tests/`](./css-tokenizer-tests/) - Comprehensive CSS tokenization test suite
- [`popular/`](./popular/) - Popular CSS frameworks and libraries
- [`postcss-parser-tests/`](./postcss-parser-tests/) - PostCSS compatibility test suite
- [`basic/`](./basic/)  - Basic CSS files for litmus/sanity checking
