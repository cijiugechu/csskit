# Packages

This directory contains packages for different platforms and integrations beyond the core Rust crates.

## Contents

- [`csskit/`](./csskit/) - npm package providing Node.js bindings for csskit
- [`csskit_vscode/`](./csskit_vscode/) - [Visual Studio Code](https://code.visualstudio.com/) extension for CSS development
- [`csskit_zed/`](./csskit_zed/) - [Zed editor](https://zed.dev/) extension for CSS development
